### Youtube Download Tool 只是一個簡單的下載工具

空閒的時候做的一個小工具

由於網路上有些免費下載軟體

只提供低解析度的影片下載

所以才想自己做一個

> 使用語言：Python

> 使用套件：pytube3 && moviepy

Before you use it,

you may need to install these two packages `pytube3` and `moviepy`.

or you can try this below↓

```
pip install -r requirement.txt
```
